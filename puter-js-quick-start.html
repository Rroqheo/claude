🚀 Puter.js 快速入门

四大核心功能演示：AI 聊天 • 云存储 • 键值存储 • 身份验证

🤖 示例1：GPT-4o AI 聊天

使用 Puter.js 调用 GPT-4o，无需 API 密钥，完全免费！

<script src="https://js.puter.com/v2/"></script> <script> puter.ai.chat('Why did the chicken cross the road?') .then(puter.print); </script>
等待 AI 响应...
☁️ 示例2：云存储

轻松实现文件的云端存储和读取，数据永久保存。

<script> async function fileDemo() { // 写入文件 await puter.fs.write('hello.txt', 'Hello, Puter!'); puter.print('File written successfully<br>'); // 读取文件 const fileContent = await puter.fs.read('hello.txt'); puter.print('File content: ', await fileContent.text()); } fileDemo(); </script>
等待文件操作...
🗃️ 示例3：键值存储

云端键值数据库，适合存储用户偏好、应用设置等。

<script> async function kvDemo() { // 设置值 await puter.kv.set('user_preference', 'dark_mode'); puter.print('Preference saved<br>'); // 获取值 const preference = await puter.kv.get('user_preference'); puter.print('User preference:', preference); } kvDemo(); </script>
等待键值操作...
🔐 示例4：身份验证

自动处理用户登录，访问云服务时自动提示认证。

<script> async function authDemo() { // 检查登录状态 const isSignedIn = puter.auth.isSignedIn(); puter.print('Is user signed in? ', isSignedIn, '<br>'); if (!isSignedIn) { // 触发登录 await puter.auth.signIn(); puter.print('User signed in successfully<br>'); } // 获取用户信息 const user = await puter.auth.getUser(); puter.print('User info:', JSON.stringify(user)); } authDemo(); </script>
等待身份验证操作...
