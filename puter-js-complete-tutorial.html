📚 Puter.js 入门介绍
🎯 什么是 Puter.js？

Puter.js 是一个革命性的无服务器平台，它将身份验证、云存储和 AI 功能直接集成到您的前端代码中。

  • 🆓 完全免费 - 对应用程序和开源项目
  • 🔑 无需 API 密钥 - 即插即用
  • 🤖 AI 功能 - GPT-4o、Claude 等
  • ☁️ 云存储 - 文件和数据管理
  • 🔐 身份验证 - 自动用户管理

采用创新的"用户付费"模式，开发者无需管理 API 密钥或担心计费问题。

⚡ 快速开始

只需一行代码即可开始使用：

<script src="https://js.puter.com/v2/"></script>

然后就可以使用全局 puter 对象：

// AI 聊天 await puter.ai.chat('你好，世界！'); // 云存储 await puter.fs.write('file.txt', '内容'); // 键值存储 await puter.kv.set('key', 'value'); // 身份验证 const user = await puter.auth.getUser();

🎯 完整应用示例

构建一个完整的 Web 应用：

class PuterApp { constructor() { this.init(); } async init() { // 检查登录状态 if (!puter.auth.isSignedIn()) { await puter.auth.signIn(); } // 加载用户设置 await this.loadSettings(); // 初始化 UI this.setupUI(); } async loadSettings() { try { const settings = await puter.kv.get('app_settings'); this.settings = JSON.parse(settings || '{}'); } catch (error) { this.settings = { theme: 'dark' }; } } async saveData(key, data) { await puter.fs.write(`${key}.json`, JSON.stringify(data, null, 2)); } async askAI(question) { return await puter.ai.chat(question); } } // 使用应用 const app = new PuterApp();
